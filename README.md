<h1 align="center">Breadth-First Search Practice</h1>

<p align="center">
A simple Python program for practicing breadth-first graph traversal.
</p>

<hr/>

<h2>📝 Description</h2>
This project demonstrates a Breadth-First Search algorithm on a graph using Python.  
The algorithm explores nodes level by level, visiting all neighboring nodes before moving deeper into the graph.  
It records the traversal order and illustrates how BFS is used for shortest-path discovery in unweighted graphs.
